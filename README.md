# الميزان - Mizan

تطبيق متطور لمتابعة أسعار الذهب والعملات العالمية والليرة السورية لحظة بلحظة. يتميز التطبيق بواجهة مستخدم عصرية وشريط أسعار متحرك تفاعلي.

## المميزات الرئيسية
- **تحديث مباشر**: جلب الأسعار من Binance ومن موقع sp-today.com لليرة السورية.
- **شريط أسعار متحرك (Ticker)**: يعرض الأسعار بشكل متحرك من اليسار إلى اليمين مع إمكانية التوقف عند التمرير.
- **دعم البيع والشراء**: عرض أسعار المبيع والشراء لكل العملات.
- **محول عملات ذكي**: حساب التحويلات بين الذهب والعملات المختلفة بدقة عالية.
- **رسوم بيانية**: عرض التغيرات التاريخية لأسعار الذهب.
- **ثنائي اللغة**: دعم كامل للعربية والإنجليزية.
- **الوضع الليلي والنهاري**: واجهة تتكيف مع تفضيلات المستخدم.

## التقنيات المستخدمة
- React.js
- Tailwind CSS
- Recharts (للرسوم البيانية)
- Binance API (للذهب والعملات العالمية)
- AllOrigins Proxy (لكشط بيانات الليرة السورية من sp-today)
